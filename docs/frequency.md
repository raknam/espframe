# Frequency

The **Frequency** settings control how often the frame advances to the next photo. All options are available in the device web UI at `http://<device-ip>/` under the **Frequency** card.

## Slideshow Interval

| Setting | Default | Description |
|---------|---------|-------------|
| **Slideshow Interval** | 15 seconds | Time each photo is displayed before advancing to the next |

Choose how long each image stays on screen. Shorter intervals give more variety; longer intervals let you enjoy each photo and reduce how often the display changes.

### Available intervals

| Option | Use case |
|--------|----------|
| **10 s** | Quick rotation, high variety |
| **15 s** | Balanced default; good for mixed content |
| **20 s** | Slightly more time per photo |
| **30 s** | Comfortable viewing without feeling rushed |
| **45 s** | Focus on each image |
| **1 min** | Calm, gallery-like pace |
| **2 min** | Minimal distraction; good for desks or waiting areas |
| **3 min** | Very slow rotation |
| **5 min** | Almost static; each photo gets long attention |
| **10 min** | Near-static display; ideal for art or single-photo focus |

Changes take effect on the next photo transition. No reboot is required.

::: tip
If the frame feels too busy, try 30 s or 1 min. If you want it to feel more like a traditional photo frame, 2–5 min works well.
:::
