# Roadmap

Planned features and improvements for Espframe. Items are roughly ordered by priority within each category.

## Image Controls

Control **which** photos the frame displays.

| Feature | Status |
|---|---|
| Filter by album | Planned |
| Filter by person / face | Planned |
| Favourite-only mode | Planned |
| Date-range filter (e.g. "this month in history") | Planned |
| Exclude videos / motion photos | Planned |
| Daily memory groups (e.g. "on this day" photo collections) | Planned |

## Offline Storage

Store images offline when server access isn't available.

| Feature | Status |
|---|---|
| Offline image cache using SD card | Planned |

## Ambient Light Adaptation

Automatically adjust display output based on ambient light conditions.

| Feature | Status |
|---|---|
| Automatic brightness based on ambient light level | Planned |
| Automatic white balance based on ambient light temperature | Planned |

## Photo Metadata

Show information about the **currently displayed photo**.

| Feature | Status |
|---|---|
| Date & location overlay | Planned |
| Camera / lens info | Planned |
| Album name | Planned |
| People / faces detected | Planned |
| EXIF details on tap | Planned |

## Screen Rotation

Support mounting the frame in landscape or portrait orientation.

| Feature | Status |
|---|---|
| Configurable screen rotation (0°/90°/180°/270°) | Planned |
| Optimized layouts for both landscape and portrait orientations | Planned |

## Networking

Remote access and VPN connectivity.

| Feature | Status |
|---|---|
| [WireGuard VPN client](./wireguard) | Planned |
| Remote access to Immich over VPN | Planned |

## Error Handling

Error handling to improve the user experience.

| Feature | Status |
|---|---|
| Graceful handling of Wi-Fi connection loss | Planned |
| Graceful handling of Immich server connection loss | Planned |

## Onscreen Settings

Add controls to customize frame settings from the screen rather than the web server.

| Feature | Status |
|---|---|
| Screen rotation | Planned |
| Firmware update controls | Planned |
| Backlight brightness | Planned |
